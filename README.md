These are the hands-on lab exercises for the WSQ Certified Lean Six Sigma Black Belt (CLSSBB) Training course delivered by [Tertiary Infotech Academy Pte Ltd](https://www.tertiarycourses.com.sg/).

This repository contains **34 guided Lean Six Sigma Black Belt labs** (34 core and 0 elective), structured around the **DMAIC roadmap** and grounded in the Council for Six Sigma Certification (CSSC) Black Belt body of knowledge.

## How to use

1. Read the Learner Guide first — it follows the same DMAIC order as the course.
2. Complete the labs in order, applying each to your OWN workplace capstone project.
3. Keep every worksheet — the final lab combines them into one improvement package.
4. Review the 'Check your work' step at the end of each lab before moving on.
